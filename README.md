
# SmartFinance

A mobile application that helps you to manage your finances.

## Run Locally

Clone the project

```bash
  git clone https://github.com/the-error4-0-4/Smart-Finance.git
```

Go to the project directory

```bash
  cd Smart-Finance
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
